# Restaurant Ordering System – Frontend

Frontend built with ReactJS for browsing the menu, placing orders, and viewing restaurant information.

## Project Info
- Version: 1.0.0
- Author: Mostafa Diab
- Technology: ReactJS
- Backend: Node.js + Express (local)

## Features
- Responsive design (desktop & mobile)
- Dynamic menu fetched from backend
- Cart and order submission
- Smooth scrolling navigation
- Admin dashboard (optional)

## Pages
- Home
- Menu
- About
- Contact
- Order
- Admin
- Login

## Setup
cd frontend
npm install
npm start

Frontend runs at:
http://localhost:3000

## Backend Connection
Frontend uses a local proxy:

"proxy": "http://localhost:5000"

API calls like /api/items are routed automatically.
